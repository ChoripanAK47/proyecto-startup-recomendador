# Proyecto de Startup Recomendador

## Descripción
Proyecto base para una plataforma de recomendación de productos/servicios basada en comportamiento de usuarios.

Incluye:
- API básica en Python (Flask)
- Contenerización con Docker
- CI/CD con GitHub Actions
- Preparación para despliegue en la nube

---

## Arquitectura
Arquitectura híbrida:
- Local: desarrollo y pruebas
- Cloud: despliegue (Render) y base de datos (Supabase)

---

## Tecnologías
- Python 3
- Flask
- Docker
- GitHub Actions
- PostgreSQL
